xlwings is an MS Excel plugin that provides access to Python from within Excel.

To make this spreadhseet work you need the install xlwings plugin. You can install it following the instructions as found on their website https://www.xlwings.org/

This .xlsm file requires macros to be allowed.

You are also likely to need to install win32api.

With an Anaconda env there might be more success with the following command to initialise your xlwings spreadsheet:

# xlwings quickstart myproject --standalone
